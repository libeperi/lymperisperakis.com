# lymperisperakis.com

Personal portfolio site, built with [Next.js](https://nextjs.org) and deployed on [Vercel](https://vercel.com).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Deployment

Pushes to `main` are deployed automatically by Vercel via the GitHub integration.
